üéÆ The Quick and Dirty Way to Update tModLoader Server

This whole process is basically downloading the new stuff and dropping it where the old stuff used to be. It’s way easier than trying to remember your Wi-Fi password.

There are two main ways to update your tModLoader server. One is super simple if you started with the Steam version. The other is a little more hands-on for everyone else.

Method 1: The Steam Way (If You Used the Steam App)

This is the “easiest button available” option, but only works if you’ve been launching your dedicated server through the actual tModLoader application on Steam.

  1. Update the Game: First, make sure your regular tModLoader game on Steam is already updated. Steam handles this automatically, but maybe give it a quick check. If you launched the game recently, you’re probably good.
  2. Run the Server from Steam: Go to your Steam Library, find tModLoader, and hit Play. When it asks what you want to do, choose the option that says “Launch Server” or “Launch Dedicated Server.”
  3. Magic Happens: Here’s the cool part: When you launch the server this way, tModLoader often checks for and updates its own server files in the background. It basically fixes itself.

But here’s the thing: If that doesn’t work, or if you’re running the server on a completely separate computer without Steam, you have to do it the old-school way. Which is still pretty easy, I promise.

Method 2: The Manual File Swap (The Real Way to Update)

This is how the pros (and now you) update the tModLoader server for real. You’re just manually replacing the engine of your server with a newer one.

Step 1: Find the Right Files

Look, the first step in any surgery is finding the right tool. In this case, it’s the right version of the server files.

  1. Check Your Game Version: Start tModLoader on your PC. Look at the bottom-right corner of the title screen. You’ll see the version number, like v2024.06.25.12. Write that down. That‚Äôs the version you need.
  2. Grab the Server Download: You need to go to the official tModLoader GitHub page. Don’t panic, it’s just a link to a file. You want to look for the version number that matches what you wrote down. It’s usually called something like tModLoader.Server.v[Your Version Number].zip. Download that file.
  3. Unzip the Files: Double-click the downloaded ZIP file. You’ll see a bunch of files inside, including the one you care most about: tModLoaderServer.exe.

Step 2: Swap the Server Files

Now you just need to drop those shiny new files into your existing server folder. It’s like upgrading the engine in your car.

  1. Find Your Server Folder: You need to know where your old server files are. If you‚Äôre running a dedicated server, you know where this is. If you’re using the standard tModLoader path, it’s usually inside your Documents\My Games\Terraria\tModLoader\Worlds folder (though the exact location depends on your setup).
  2. Delete the Old Files: Find your existing tModLoaderServer.exe and any other old files that came with it. Delete them. Yeah, really. You don’t want old parts mixing with new ones.
  3. Copy the New Files: Take ALL the files you unzipped from the download and copy them into your server folder.

That’s it. You just replaced the old, outdated server program with the new one.

Step 3: Start the Updated Server

Now you just launch the new program like you always did.

  • Find and double-click the new tModLoaderServer.exe file in your server folder.
  • The server will start up, and you’ll see the new version number pop up in the command window.

Quick Reality Check: If you did this right, the version number on your server screen and the version number on your game‚Äôs title screen should be exactly the same. If they match, you’re good to go!

üõë What If You Messed Up? (Troubleshooting)

Hey, it happens. If your server is still being a pain, here are the most common ways people screw this up.

“My Server and Client Versions Still Don’t Match!”

This is 99% of the problems.

  • You Downloaded the Wrong Version: Did you download the exact version number you saw on your game’s title screen? If your game is v2024.07.01.1 and your server is v2024.06.25.12, it won’t work. Go back and check those numbers.
  • You Missed a File: Did you copy all the files from the ZIP? Sometimes there are important files alongside the main .exe. Copy them all. It’s safer.

“Now My Mods Are Gone!”

Don’t worry, your mods should be safe.

  • Server Mods: Server mods are stored in a different folder, usually in Documents\My Games\Terraria\tModLoader\Mods. Updating the server program (the .exe file) doesn’t touch your mod files. They’re still there.

Bottom line: The server program and the server’s mod files are in two different places. You only replaced the program. Everything else is fine.
