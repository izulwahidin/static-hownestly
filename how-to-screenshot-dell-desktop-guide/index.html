Automatic Save to Your Pictures Folder

Know what’s better than copying an image? Saving it automatically as a file. This shortcut takes the picture and saves it for you in one step. Shocking, I know.

  1. Press and Hold: Find the Windows Key (it looks like a little window flag).
  2. Tap the Shot: While holding the Windows Key, quickly press the PrtScn (Print Screen) key. It might be labeled “Prnt Scrn” or “Prt Sc.” You’ll usually find it up near your F1-F12 keys.

  4. Find the File: The screenshot is now saved as a PNG file in your Pictures folder, in a subfolder called Screenshots.
